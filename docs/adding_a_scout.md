# Adding a New Scout

1. Create a new folder:

scouts/<brand>

2. Create:

collector.py
parser.py
rules.py
scorer.py
__init__.py

3. Create matching knowledge.

knowledge/<brand>.py

4. Create matching memory.

memory/<brand>_memory.py

5. Create matching patterns.

patterns/<brand>_patterns.py

No changes are required to Atlas Brain.

No changes are required to the Reasoning Engine.

No changes are required to Signals.

Atlas automatically supports the new scout.
